# Stone

Stone is a Brazilian financial technology company offering REST APIs for payment processing, Pix instant payments, digital banking accounts, boleto issuance, credit products, payment links, and stone-terminal management for merchants. Their platform enables businesses to authorize, capture, cancel, and query transactions across online and physical sales channels.

## APIs

- **Stone Online API** - Transactional API for authorizing, capturing, canceling, and querying card payment transactions for e-commerce and physical POS environments.
- **Stone OpenBank API** - Open banking REST API for digital account management, Pix payments, boleto issuance, payment links, QR codes, KYC, and recharges.
- **Stone Connect API** - Integration API for connecting POS systems to Stone payment terminals.
- **Stone Conciliation API** - Backoffice API for financial reconciliation, sales monitoring, and transaction reporting.
